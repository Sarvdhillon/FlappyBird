# Pixel Bird - Premium Edition


Create a modern and responsive Flappy Bird-inspired browser game using HTML, CSS, and JavaScript.

### Requirements

- Build the game using:
  - HTML5 Canvas
  - CSS3
  - Vanilla JavaScript

- Include:
  - Start screen
  - Game over screen
  - Score tracking system
  - Best score using localStorage
  - Smooth bird movement and gravity physics
  - Pipe obstacle generation
  - Collision detection
  - Sound effects
  - Particle effects
  - Responsive full-screen layout

- Design Requirements:
  - Pixel-art inspired theme
  - Smooth animations and transitions
  - Modern UI with glassmorphism effects
  - Mobile and desktop compatibility

- Controls:
  - Spacebar to jump
  - Mouse/touch support for mobile devices

- Additional Features:
  - Dynamic background
  - Screen shake effect on collision
  - Animated score effects
  - Retry functionality

### Technologies Used

- HTML5
- CSS3
- JavaScript
